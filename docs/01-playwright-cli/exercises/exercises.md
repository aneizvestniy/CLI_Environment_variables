# Вправи — 01 Playwright CLI

## Вправа 1 — Фільтрація запуску

1) Запусти всі тести
2) Запусти один файл
3) Запусти тест по `--grep`
4) Запусти тест у headed режимі

Запиши команди у відповідь.

## Вправа 2 — Debug

1) Запусти будь-який тест через:

```bash
npx playwright test --debug
```

2) Дійди до моменту, де тест виконується, і зупинись на кроці.

Опиши:
- що показує Inspector
- як знайти локатор
- як продовжити виконання
