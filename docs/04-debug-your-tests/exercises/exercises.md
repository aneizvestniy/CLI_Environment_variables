# Вправи — 04 Debug your tests

## Вправа 1 — Inspector

1) Запусти будь-який тест через:

```bash
npx playwright test --debug
```

2) Зупини виконання на кроці.
3) Знайди локатор елемента через picker.

Опиши, як ти:
- крокуєш
- змінюєш локатор
- продовжуєш виконання

## Вправа 2 — page.pause()

1) Додай `page.pause()` у тест.
2) Запусти у headed/debug режимі.

Поясни, чому `page.pause()` не працює у headless.
