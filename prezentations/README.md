# Презентації

Ця папка містить презентації у форматі Markdown (Marp-сумісні), які відповідають модулям/темам цього репозиторію.

## Конвенція

- Іменування файлів: `NN-topic.md` (наприклад: `01-cli-basics.md`).
- 1 файл = 1 презентація.
- Експорт у PDF/PPTX можна робити через Marp або інші інструменти.
