# Playwright CLI

CLI, або інтерфейс командного рядка (Command Line Interface), є способом взаємодії з комп'ютером за допомогою текстових команд. Він дозволяє користувачеві виконувати операції, керувати системою, запускати програми та виконувати різноманітні завдання шляхом введення команд безпосередньо в командний рядок.

## Основні компоненти CLI

- **Команди**: інструкції, які вказують системі, яку дію виконати.
- **Аргументи**: дані, які надсилаються разом з командою і уточнюють, що саме робити.
- **Опції**: додаткові параметри, які змінюють поведінку команди.
- **Параметри**: налаштування інтерфейсу командного рядка (шрифт, колір, розмір вікна тощо).

CLI використовується для різних завдань — від керування операційною системою до автоматизації задач, розробки програм і роботи з веб‑серверами.

## Найбільш поширені команди Playwright CLI

### Запуск тестів

- Запуск усіх тестів:

```bash
npx playwright test
```

- Запуск одного файлу тесту:

```bash
npx playwright test tests/todo-page.spec.ts
```

- Запуск групи файлів тестів:

```bash
npx playwright test tests/todo-page/ tests/landing-page/
```

- Запуск файлів, які містять `my-spec` або `my-spec-2` у назві:

```bash
npx playwright test my-spec my-spec-2
```

- Запуск тестів, які знаходяться у рядку 42 в `my-spec.ts`:

```bash
npx playwright test my-spec.ts:42
```

- Запуск тесту за назвою:

```bash
npx playwright test -g "додати пункт завдання"
```

### Режими запуску

- Запуск тестів у браузері з інтерфейсом:

```bash
npx playwright test --headed
```

- Запуск усіх тестів у вказаному проекті:

```bash
npx playwright test --project=chromium
```

- Вимкнення паралельного виконання:

```bash
npx playwright test --workers=1
```

- Вибір репортера:

```bash
npx playwright test --reporter=dot
```

- Режим налагодження з Playwright Inspector:

```bash
npx playwright test --debug
```

- Інтерактивний режим (UI mode):

```bash
npx playwright test --ui
```

### Довідка

- Виведення команд доступних в CLI:

```bash
npx playwright test --help
```

Більше опцій можна подивитись на сторінці офіційної документації.
